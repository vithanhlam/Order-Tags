Easily assign tags to orders and customize unlimited colors effortlessly.

== Description ==

This application easily assigns tags to orders, and you can effortlessly customize unlimited colors.

Advantages include the ease of quickly adding and removing tags with the ability to select custom colors that appear alongside order information.

Support https://code-wp.com

== Installation ==

1. Upload the plugin files to the /wp-content/plugins/ directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the ‘Plugins’ screen in WordPress
3. Setting in Woocommerce -> Tags Order
